
# CLAEX

CLAEX é um organizador de grades horarias automatico, seu objetivo é facilitar a gestão e divulgação das mesmas

[Link do projeto](https://udanidan.github.io/claex/)

## Qual o diferencial

CLAEX é um sistema que interliga varias escolas diferentes, podendo verificar automaticamente se um professor está ocupado em determinado horario e permitindo solicitar trocas evitando erros de comunicação e facilitando a comunicação

- Mais organizado
- Menos erros
- Mais facil
